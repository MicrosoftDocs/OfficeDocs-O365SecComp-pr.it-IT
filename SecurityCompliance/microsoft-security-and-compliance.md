# <a name="get-ready-for-the-new-microsoft-365-security-center-and-microsoft-365-compliance-center"></a>Prepararsi per il nuovo Centro protezione di Microsoft 365 e centro conformità Microsoft 365

**Siamo lieti di annunciare la nuova [interfaccia sicurezza Microsoft 365](#microsoft-365-security-center) e [centro conformità Microsoft 365](#microsoft-365-compliance-center)distribuzione partendo dalla fine di gennaio 2019 e tramite 2019 marzo**. In questo articolo viene fornita una panoramica delle novità, [cosa aspettarsi](#what-to-expect)e [licenze necessarie e le autorizzazioni necessarie](#required-licenses-and-permissions).

## <a name="microsoft-365-security-center"></a>Centro protezione di Microsoft 365

Il nuovo Centro protezione Microsoft 365 include un dashboard che consentono di gestire e monitorare la sicurezza tra l'identità, dati, dispositivi, App e dell'infrastruttura. È inoltre possibile accedere facilmente la nuova Microsoft sicura punteggio, nuovi dispositivi threat report, nuovi report di rischio di identità e i rapporti di protezione di applicazione Cloud. 

![Nuovo Centro protezione di Microsoft 365](media/m365-security-center.png)

Quando si utilizza il Centro protezione 365 Microsoft per la prima volta, vedrai informazioni nella parte superiore dello schermo per facilitare l'avvio. Si vedrà inoltre come esplorare facilmente le funzionalità di protezione che interessano più esplorazione.

Una volta il Centro protezione di Microsoft 365 è abilitato per il tenant, sarà in grado di accedervi in [https://security.microsoft.com](https://security.microsoft.com). 

> [!NOTE]
> Deve essere assegnato un ruolo di Azure Active Directory valido per accedere al Centro protezione di Microsoft 365. Per ulteriori informazioni, vedere la sezione [necessarie autorizzazioni e le licenze](#required-licenses-and-permissions) (in questo articolo).

## <a name="microsoft-365-compliance-center"></a>Centro conformità Microsoft 365

Il nuovo centro conformità Microsoft 365 fornisce visibilità in conformità gestione Microsoft, che riflette le condizioni generali conformità generale e offre azioni che consentono di configurare le impostazioni per soddisfare gli obblighi di conformità complessi consigliate. È necessario facile accesso a etichette di riservatezza e conservazione e criteri, prevenzione della perdita di dati (DLP), governance di dati, eDiscovery, le richieste di oggetto di dati (dettagliata), la gestione dei casi e sicurezza App Cloud. Inoltre, è possibile ottenere approfondimenti e utilizza la protezione automazione per ridurre i rischi per la conformità e salvaguardia del patrimonio immobiliare digitale. 

![Centro conformità Microsoft 365](media/m365-compliance-center.png)

Quando si utilizza il centro conformità 365 Microsoft per la prima volta, vedrai informazioni nella parte superiore dello schermo per facilitare l'avvio. Verrà visualizzato come esplorare facilmente le funzionalità di conformità che interessano più esplorazione.

Una volta centro conformità Microsoft 365 è abilitato per il tenant, sarà di accedervi in [https://compliance.microsoft.com](https://compliance.microsoft.com).  

> [!NOTE]
> Deve essere assegnato un ruolo di Azure Active Directory valido per accedere al centro conformità Microsoft 365. Per ulteriori informazioni, vedere la sezione [necessarie autorizzazioni e le licenze](#required-licenses-and-permissions) (in questo articolo).

## <a name="what-to-expect"></a>Cosa aspettarsi

### <a name="coming-soon"></a>Prossimamente!

Il nuovo Centro protezione di Microsoft 365 e il nuovo centro conformità Microsoft 365 sono ripristino dello stato fuori inizio alla fine di gennaio e tramite 2019 marzo. È necessario avere accesso per il nuovo Centro protezione di Microsoft 365 e Microsoft 365 centro conformità all'interno di questo intervallo di tempo.

### <a name="easy-access"></a>Per accedere facilmente!

Spostamento migliorato, soluzioni integrate e semplificare l'operazione, sarà in grado di vedere e accedere alle informazioni interessano maggiormente e sfruttare la sicurezza potente e soluzioni di conformità in Microsoft 365.

### <a name="smooth-transition"></a>Transizione!

È possibile ottenere una transizione senza problemi ai centri di nuovo. Dopo la modifica è completamente di implementazione, si prevede di ritirare precedente 365 Microsoft Security & centro conformità ([https://protection.microsoft.com](https://protection.microsoft.com)). Consente di modificare l'esperienza dell'amministratore, ma ciò non influisce le configurazioni di conformità e la sicurezza corrente.

Dopo l'aggiornamento è stato implementato, se l'organizzazione dispone di Microsoft 365 aziendale E3 o E5, possono agli amministratori di sicurezza e conformità:

- Consente di passare direttamente a [https://security.microsoft.com](https://security.microsoft.com) e [https://compliance.microsoft.com](https://compliance.microsoft.com); <br>oppure  
- Accedere all'interfaccia di amministrazione di Microsoft 365 e quindi passare il nuovo Centro protezione di Microsoft 365 e centro conformità Microsoft 365 (collegamenti in interfacce di amministrazione nel riquadro di spostamento a sinistra).

> [!TIP]
> Se si utilizza la protezione di Office 365 & centro conformità ([https://protection.office.com](http://protection.office.com)), sarà comunque possibile configurare e gestire le impostazioni di Office 365 all'interno del & di sicurezza di Office 365 esistente centro conformità. Le configurazioni verranno mantenute in esistente protezione di Office 365 & centro conformità anche come il nuovo Centro protezione di Microsoft 365 e centro conformità Microsoft 365.  

## <a name="required-licenses-and-permissions"></a>Le autorizzazioni e le licenze necessarie

L'organizzazione deve essere Microsoft 365 aziendale E3 o E5 per ottenere il nuovo Centro protezione di Microsoft 365 e centro conformità Microsoft 365.

Gli utenti devono essere assegnati il ruolo amministratore globale, amministratore di conformità, amministratore della sicurezza o lettore di sicurezza in Azure Active Directory per accedere al nuovo Centro protezione di Microsoft 365 o centro conformità Microsoft 365.

- Gli amministratori globali possono accedere al Centro protezione sia nel centro conformità

- Gli amministratori di conformità possono accedere al centro conformità

- Security Administrators o sicurezza lettori possono accedere al Centro protezione

> [!NOTE]
> Ruoli aggiuntivi, tra cui operatore di protezione e amministrazione di dati di conformità, sono disponibile a breve.
