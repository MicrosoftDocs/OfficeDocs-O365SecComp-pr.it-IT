## <a name="recover-an-inactive-mailbox"></a><span data-ttu-id="365ea-127">Recuperare una cassetta postale inattiva</span><span class="sxs-lookup"><span data-stu-id="365ea-127">Recover an inactive mailbox</span></span>

    ```

    ```
   
    > [!IMPORTANT]
    > <span data-ttu-id="365ea-p107">Nel comando precedente, usare il valore della proprietà **DistinguishedName** o **ExchangeGUID** per identificare la cassetta postale inattiva.
