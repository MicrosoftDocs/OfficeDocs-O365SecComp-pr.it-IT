<span data-ttu-id="040bf-p114">SharePoint Online archivi BLOB binari file (ad esempio, i flussi di file) in Microsoft Azure. Finally, SharePoint Online has real-time monitoring in place to detect when an HTTP request reads or writes data for more than one tenant.
