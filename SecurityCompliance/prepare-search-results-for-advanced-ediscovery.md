Advanced eDiscovery features include:</span></span>
  
- <span data-ttu-id="8fe6f-p102">**Riconoscimento ottico dei caratteri** -quando si preparano i risultati della ricerca per Advanced eDiscovery, la funzionalità OCR (Optical Character Recognition) estrae automaticamente il testo dalle immagini e lo include con i risultati di ricerca caricati in Advanced eDiscovery per l'analisi. This allows you to apply the text analytic capabilities of Advanced eDiscovery (near-duplicates, email threading, themes, and predictive coding) to the text content in image files. Advanced eDiscovery OCR supports the following formats for image files:</span></span>

    - <span data-ttu-id="8fe6f-110">GIF</span><span class="sxs-lookup"><span data-stu-id="8fe6f-110">GIF</span></span>
    - <span data-ttu-id="8fe6f-111">JPEG</span><span class="sxs-lookup"><span data-stu-id="8fe6f-111">JPEG</span></span>

- <span data-ttu-id="8fe6f-p104">**Threading della posta elettronica** -consente di identificare i messaggi univoci in un thread di posta elettronica in modo da poter concentrarsi solo sulle nuove informazioni in ogni messaggio. - <span data-ttu-id="8fe6f-p105">**Temi** -informazioni utili per ottenere informazioni importanti sui dati oltre alle statistiche di ricerca di parole chiave. - <span data-ttu-id="8fe6f-p106">**Codifica predittiva** : consente di formare il sistema su quello che si sta cercando, consentendo di prendere decisioni (se qualcosa è pertinente o meno) su un set di documenti di piccole dimensioni. - <span data-ttu-id="8fe6f-p107">**Esportazione dei dati per le applicazioni di revisione** : è possibile esportare i dati da Advanced eDiscovery e Office 365 dopo aver completato l'analisi e aver ridotto il set di dati.
