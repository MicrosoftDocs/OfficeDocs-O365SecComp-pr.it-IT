## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="ab333-110">Come vengono identificati i messaggi duplicati</span><span class="sxs-lookup"><span data-stu-id="ab333-110">How duplicate messages are identified</span></span>

<span data-ttu-id="ab333-111">Gli strumenti di eDiscovery di Office 365 utilizzano una combinazione delle proprietà di posta elettronica seguenti per determinare se un messaggio è un duplicato:</span><span class="sxs-lookup"><span data-stu-id="ab333-111">Office 365 eDiscovery tools use a combination of the following email properties to determine whether a message is a duplicate:</span></span>
  
- <span data-ttu-id="ab333-112">**InternetMessageId** -questa proprietà consente di specificare l'identificatore del messaggio Internet di un messaggio di posta elettronica, che è un identificatore univoco globale che fa riferimento a una versione specifica di un messaggio specifico.</span><span class="sxs-lookup"><span data-stu-id="ab333-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="ab333-113">Questo ID è generato dal programma client di posta elettronica del mittente o dal sistema di posta elettronica host che invia il messaggio.</span><span class="sxs-lookup"><span data-stu-id="ab333-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="ab333-114">Se un utente invia un messaggio a più di un destinatario, l'ID del messaggio Internet sarà lo stesso per ogni istanza del messaggio.</span><span class="sxs-lookup"><span data-stu-id="ab333-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="ab333-115">Le revisioni successive al messaggio originale riceveranno un identificatore di messaggio diverso.</span><span class="sxs-lookup"><span data-stu-id="ab333-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="ab333-116">**ConversationTopic** -la sua proprietà specifica l'oggetto del thread di conversazione di un messaggio.</span><span class="sxs-lookup"><span data-stu-id="ab333-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="ab333-117">Il valore della proprietà **ConversationTopic** è la stringa che descrive l'argomento generale della conversazione.</span><span class="sxs-lookup"><span data-stu-id="ab333-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="ab333-118">Una conservazione è costituita da un messaggio iniziale e da tutti i messaggi inviati in risposta al messaggio iniziale.</span><span class="sxs-lookup"><span data-stu-id="ab333-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="ab333-119">I messaggi all'interno della stessa conversazione hanno lo stesso valore per la proprietà **ConversationTopic** .</span><span class="sxs-lookup"><span data-stu-id="ab333-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="ab333-120">Il valore di questa proprietà è in genere la riga dell'oggetto del messaggio iniziale che ha generato la conversazione.</span><span class="sxs-lookup"><span data-stu-id="ab333-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span> 
    
- <span data-ttu-id="ab333-121">**BodyTagInfo** -questa è una proprietà di archivio di Exchange interna.</span><span class="sxs-lookup"><span data-stu-id="ab333-121">**BodyTagInfo** - This is an internal Exchange store property.</span></span> <span data-ttu-id="ab333-122">Il valore di questa proprietà viene calcolato controllando vari attributi nel corpo del messaggio.</span><span class="sxs-lookup"><span data-stu-id="ab333-122">The value of this property is calculated by checking various attributes in the body of the message.</span></span> <span data-ttu-id="ab333-123">Questa proprietà viene utilizzata per identificare le differenze nel corpo dei messaggi.</span><span class="sxs-lookup"><span data-stu-id="ab333-123">This property is used to identify differences in the body of messages.</span></span> 
    
<span data-ttu-id="ab333-124">Durante il processo di esportazione di eDiscovery, queste tre proprietà vengono confrontate per ogni messaggio che corrisponde ai criteri di ricerca.</span><span class="sxs-lookup"><span data-stu-id="ab333-124">During the eDiscovery export process, these three properties are compared for every message that matches the search criteria.</span></span> <span data-ttu-id="ab333-125">Se queste proprietà sono identiche per due (o più) messaggi, i messaggi vengono considerati duplicati e il risultato è che verrà esportata solo una copia del messaggio se la deduplicazione è abilitata.</span><span class="sxs-lookup"><span data-stu-id="ab333-125">If these properties are identical for two (or more) messages, those messages are determined to be duplicates and the result is that only one copy of the message will be exported if de-duplication is enabled.</span></span> <span data-ttu-id="ab333-126">Il messaggio esportato è noto come "elemento di origine".</span><span class="sxs-lookup"><span data-stu-id="ab333-126">The message that is exported is known as the "source item".</span></span> <span data-ttu-id="ab333-127">Le informazioni sui messaggi duplicati sono incluse nei report **results. csv** e **manifest. XML** inclusi nei risultati della ricerca esportati.</span><span class="sxs-lookup"><span data-stu-id="ab333-127">Information about duplicate messages is included in the **Results.csv** and **Manifest.xml** reports that are included with the exported search results.</span></span> <span data-ttu-id="ab333-128">Nel file **results. csv** viene identificato un messaggio duplicato che presenta un valore nella colonna **Duplica per elemento** .</span><span class="sxs-lookup"><span data-stu-id="ab333-128">In the **Results.csv** file, a duplicate message is identified by having a value in the **Duplicate to Item** column.</span></span> <span data-ttu-id="ab333-129">Il valore di questa colonna corrisponde al valore nella colonna **identità elemento** per il messaggio esportato.</span><span class="sxs-lookup"><span data-stu-id="ab333-129">The value in this column matches the value in the **Item Identity** column for the message that was exported.</span></span> 
  
<span data-ttu-id="ab333-130">Nella grafica seguente viene illustrato il modo in cui i messaggi duplicati vengono visualizzati nei report **results. csv** e **manifest. XML** esportati con i risultati della ricerca.</span><span class="sxs-lookup"><span data-stu-id="ab333-130">The following graphics show how duplicate messages are displayed in the **Results.csv** and **Manifest.xml** reports that are exported with the search results.</span></span> <span data-ttu-id="ab333-131">Questi rapporti non includono le proprietà di posta elettronica descritte in precedenza, che vengono utilizzate nell'algoritmo di deduplicazione.</span><span class="sxs-lookup"><span data-stu-id="ab333-131">These reports don't include the email properties previously described, which are used in the de-duplication algorithm.</span></span> <span data-ttu-id="ab333-132">Al contrario, i report includono la proprietà dell' **identità dell'elemento** assegnata agli elementi dall'archivio di Exchange.</span><span class="sxs-lookup"><span data-stu-id="ab333-132">Instead, the reports include the **Item Identity** property that is assigned to items by the Exchange store.</span></span>

## <a name="limitations-of-the-de-duplication-algorithm"></a><span data-ttu-id="ab333-139">Limitazioni dell'algoritmo di deduplicazione</span><span class="sxs-lookup"><span data-stu-id="ab333-139">Limitations of the de-duplication algorithm</span></span>

<span data-ttu-id="ab333-140">Esistono alcune limitazioni note dell'algoritmo di deduplicazione che possono causare la marcatura degli elementi univoci come duplicati.</span><span class="sxs-lookup"><span data-stu-id="ab333-140">There are some known limitations of the de-duplication algorithm that might cause unique items to get marked as duplicates.</span></span> <span data-ttu-id="ab333-141">È importante comprendere queste limitazioni in modo da poter decidere se utilizzare o meno la funzionalità di deduplicazione facoltativa.</span><span class="sxs-lookup"><span data-stu-id="ab333-141">It's important to understand these limitations so you can decide whether or not to use the optional de-duplication feature.</span></span>
  
<span data-ttu-id="ab333-142">Esiste una situazione in cui la funzionalità di deduplicazione potrebbe identificare un messaggio come duplicato e non esportarlo (ma citarlo come duplicato nei report di <span data-ttu-id="ab333-149">Si noti che i messaggi univoci possono essere contrassegnati anche come duplicati quando la funzionalità di protezione delle pagine copy-on-Write è abilitata, come nel caso di una cassetta postale che si trova sul blocco per controversia legale o sul posto.</span><span class="sxs-lookup"><span data-stu-id="ab333-149">Note that unique messages can also be marked as duplicates when the Copy-on-Write page protection feature is enabled, as in the case of a mailbox being on Litigation Hold or In-Place Hold.</span></span> <span data-ttu-id="ab333-150">La funzionalità copy-on-Write copia il messaggio originale (e lo salva nella cartella Versions della cartella elementi ripristinabili dell'utente) prima che venga salvata la revisione all'elemento originale.</span><span class="sxs-lookup"><span data-stu-id="ab333-150">The Copy-on-Write feature copies the original message (and saves it in the Versions folder of the user's Recoverable Items folder) before the revision to original item is saved.</span></span> <span data-ttu-id="ab333-151">In questo caso, la copia riveduta e il messaggio originale (nella cartella elementi ripristinabili) potrebbero essere considerati come messaggi duplicati e quindi solo uno di essi verrebbe esportato.</span><span class="sxs-lookup"><span data-stu-id="ab333-151">In this case, the revised copy and the original message (in the Recoverable Items folder) might be considered as duplicate messages and therefore only one of them would be exported.</span></span>

> <span data-ttu-id="ab333-152">Se le limitazioni dell'algoritmo di deduplicazione possono influire sulla qualità dei risultati della ricerca, non è necessario abilitare la deduplicazione quando si esportano gli elementi.</span><span class="sxs-lookup"><span data-stu-id="ab333-152">If the limitations of the de-duplication algorithm might impact the quality of your search results, then you shouldn't enable de-duplication when you export items.</span></span> <span data-ttu-id="ab333-153">Se non è probabile che le situazioni descritte in questa sezione siano un fattore nei risultati di ricerca e si desidera ridurre il numero di elementi più probabili per essere duplicati, è consigliabile abilitare la deduplicazione.</span><span class="sxs-lookup"><span data-stu-id="ab333-153">If the situations described in this section are unlikely to be a factor in your search results, and you want to reduce the number of items most likely to be duplicates, then you should consider enabling de-duplication.</span></span>
