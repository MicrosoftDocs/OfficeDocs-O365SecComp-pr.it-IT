### <a name="scoping-an-ip-allow-list-exception-for-a-specific-domain"></a><span data-ttu-id="6ea3d-169">Ambito delle eccezioni dell'elenco di indirizzi IP consentiti per un dominio specifico</span><span class="sxs-lookup"><span data-stu-id="6ea3d-169">Scoping an IP Allow list exception for a specific domain</span></span>

<span data-ttu-id="6ea3d-p118">Ad esempio, si supponga che vi siano tre domini: ContosoA.com, ContosoB.com e ContosoC.com, e si desideri aggiungere l'indirizzo IP (per semplicità utilizzeremo 1.2.3.4) e ignorare il filtro solo per il dominio ContosoB.com. This results in the message being rescanned for all domains associated with the IP address except for ContosoB.com which is the domain listed as the exception in the rule. ContosoB.com still has an SCL of -1 which means skip filtering, whereas ContosoA.com and ContosoC.com have SCLs of 0, meaning they will be rescanned by the content filter.</span></span>
